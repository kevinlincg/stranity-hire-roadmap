# Frontend Learning Guide (React)

## 學習目標

你目前有 React Native 經驗，但 React Web 與 React Native 在 production 開發上仍有許多差異。

這份文件的目標是幫助你建立：

- React Web 基礎能力
- Component architecture 思維
- State management 能力
- Production frontend mindset

---

# 第一階段：React Fundamentals

## React Hooks

請熟悉：

- useState
- useEffect
- useMemo
- useCallback
- useRef

---

## Component Architecture

理解：

- reusable component
- state flow
- props
- component separation

---

## Routing

請學習：

- React Router
- nested route
- protected route

---

# 第二階段：State Management

## Zustand

請學習：

- global state
- store design
- shared state management

---

## React Query

請理解：

- API caching
- loading state
- mutation
- query invalidation

---

# 第三階段：UI Engineering

## Ant Design

請熟悉：

- Table
- Form
- Modal
- Drawer
- Tabs

---

## TailwindCSS

請理解：

- spacing
- layout
- responsive design

---

# 第四階段：Production Frontend

## API Integration

請熟悉：

- axios
- token handling
- interceptor
- error handling

---

## Frontend Architecture

請理解：

- feature-based architecture
- folder structure
- reusable hooks

---

## Performance

請初步理解：

- unnecessary re-render
- memoization
- list optimization

---

# 入職前最低要求

你至少應該能：

- 使用 React 建立 CRUD page
- 串接 backend API
- 使用 Ant Design 建立表單與 table
- 處理 loading/error state
- 做基本 component 拆分
