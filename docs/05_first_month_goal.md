# First Month Goal

## 目標

第一個月的核心目標是：

- 熟悉 codebase
- 理解產品流程
- 建立 debugging 能力
- 能完成小型功能與 bug fix

---

# 技術目標

## Backend

希望你能：

- 閱讀現有 API flow
- 修改簡單 API
- 新增簡單 CRUD endpoint
- trace request flow
- debug backend issue

---

## Frontend

希望你能：

- 修改 existing page
- 串接 API
- 修 UI bug
- 理解 component structure

---

# 工作內容

建議先從：

- admin tools
- table UI
- CRUD page
- bug fix
- validation

開始。

---

# 不建議一開始碰的內容

- trading engine
- high concurrency system
- websocket core
- architecture refactor

---

# 第一個月評估重點

## 1. Debug 能力

是否會：

- 看 log
- trace request
- reproduce issue

---

## 2. 問問題能力

好的問題應該：

- 有整理過
- 有 context
- 有嘗試過

---

## 3. Code Quality

是否：

- naming 清楚
- 結構清楚
- 可讀性好

---

# 第一個月 Deliverables

希望你能完成：

- 至少數個小型 PR
- 一個完整小功能
- 能獨立修 bug
